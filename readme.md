# gist-storage

## Submodule

### Clone

```sh
git clone --recurse-submodules git@github.com:panachainy/gist-storage.git
```

### Aadd

```sh
git submodule add -b master <git-url>
```

### Fix branch submodule

```sh
git config -f .gitmodules submodule.<submodule-repo-name>.branch master
```
